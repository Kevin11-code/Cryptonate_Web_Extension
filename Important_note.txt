Compile native python file
first add https://github.com/pyinstaller/pyinstaller/blob/a11e82681b697b36b2690ffa8365acd130b8885d/PyInstaller/hooks/hook-xgboost.py file in
PYTHONHOME/lib/site-package/_pyinstaller_hooks_contrib/hooks/stdhooks or 
create a new python file named hook-xgboost.py at PYTHONHOME/lib/site-package/_pyinstaller_hooks_contrib/hooks/stdhooks with following content in it
#-----------------------------------------------------------------------------

datas, binaries, hiddenimports = collect_all("xgboost")



then run pyinstaller --onefile --noconfirm -F native_host7.py --collect-all "xgboost"
